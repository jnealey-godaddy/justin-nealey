Airo for WordPress FAQ

What is Airo for WordPress?

Airo for WordPress is GoDaddy's AI-powered entry point into the WordPress ecosystem. Describe your business in a chat, and Airo builds a real WordPress site, installs the right plugins, and lets you keep improving it through conversation or the native WordPress editor.

How is this different from regular Managed WordPress Hosting?

Same hosting foundation, different experience. Airo for WordPress adds an AI layer on top of Managed WordPress Hosting so you can prompt a complete site, install plugins by asking, and keep editing through conversation. If you already know WordPress and just want hosting, our Managed WordPress Hosting page is the right starting point.

Is this a real WordPress site, or a closed-builder imitation?

Real WordPress. Every site is built on WordPress core, uses the native block editor, and runs on GoDaddy Managed WordPress Hosting. You can install any of the 60,000+ WordPress plugins, use WooCommerce, add custom code or themes, and export or move your site anytime.

Do I have to use AI for every edit?

No. You can edit your site directly in the native WordPress block editor whenever you want. Use AI when it helps, use WordPress when it doesn't, your choice, every time.

How fast are Airo for WordPress sites?

Up to 2x faster than sites not built with Airo for WordPress, based on internal GoDaddy RUM data across the Managed WordPress fleet. That is because Airo builds on the native WordPress block editor, not a heavy page builder, so pages stay lightweight and load only the assets they actually need.

Do Airo for WordPress sites use a page builder like Elementor or Divi?

No. Airo for WordPress builds on the native WordPress block editor (Gutenberg), extended with the open-source DesignSetGo plugin for advanced design blocks. This keeps sites fast and outputs clean, semantic HTML.

Can Airo write blog posts for me?

Yes. Draft full blog posts from a single prompt, refresh existing posts, retitle, or expand through conversation. Then refine in the block editor or with more AI, whatever you prefer.

Can agencies and developers customize what Airo builds?

Yes. Everything Airo builds is standard WordPress: blocks, posts, pages, and plugins. You can edit any of it directly, add custom themes or code, or hand the site off to a developer. No proprietary builder to learn, no lock-in.

Does this include WooCommerce?

Yes. WooCommerce is included and can be configured by Airo automatically when you describe a business that sells products, services, or bookings. Advanced WooCommerce features may require Deluxe or higher, pending final plan confirmation.

Is there a free trial?

Yes. 7-day free trial, no credit card required. Try the full onboarding and editing experience. Pay only when you're ready to publish or when your trial ends.

Do I own my site and data?

Yes, completely. It's real WordPress, portable on WordPress standards, with full export of your content, database, and media. No lock-in, ever.
